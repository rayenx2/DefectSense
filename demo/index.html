What it does

Production-grade MLOps pipeline

🧠

One-Class Learning (PaDiM)

Trains on normal images only — zero defect labels needed. Fits a multivariate Gaussian per ResNet18 patch position, scores via Mahalanobis distance.

⚡

ONNX Runtime Inference

2–3× CPU speedup over PyTorch. Fused ops + AVX2 SIMD. 15 MB model vs 1 GB+ torch install. 43 FPS throughput on standard CPU.

📊

MLflow Experiment Tracking

Every training run logs backbone, AUROC, training time, and artifacts. Two-layer model registry: MLflow history + JSON runtime registry.

📉

Data Drift Monitoring

PSI (Population Stability Index) computed on pixel stats at each inference. Alerts when PSI > 0.25 — detects camera drift, lighting changes, new variants.

🗂

Model Registry

Promote models: staging → production → archived. One-click rollback. Full audit log. Export registry JSON. All in the React UI.

☸️

Kubernetes Ready

HPA scales 2→10 pods on CPU > 70%. Rolling deployments for zero-downtime model swaps. AKS manifests + Azure ML pipeline included.
